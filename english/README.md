# つたわる英語 💬

「文法的に正しい英語」ではなく、**相手の意図をくみ取って“伝わる英語”** を学ぶ、日常英会話の学習アプリです。

例えば `Do you have a pen?`（ペンを持っていますか？）に対して、学校で習うのは
`Yes, I have a pen.` ですが、相手の意図は「ペンを**貸してほしい**」。
だから実際は `Sure! You can use this one.`（どうぞ、これ使ってください）と返すほうが自然に会話が成立します。

このアプリは、1つの場面に対して次の **3点セット** を並べて見せます。

1. 🏫 **学校で習う返し方**（文法的にはOK）
2. 💡 **実際に伝わる返し方**（おすすめ）
3. 📝 **なぜそう返すのか**（ポイント解説）

## 主な機能

| 機能 | 内容 |
|---|---|
| 🎭 キャラと話す | エミリー（友達）・トム（カフェ店員）・グレースさん（近所のおばあちゃん）の3人と、台本式の会話を最初から最後まで進められます。「こんにちは」から会話が始まり、毎ターン学校式/伝わる式を選んで進行。API不要・完全無料。 |
| 🔎 自由入力 | 日本語でも英語でも入力OK。用意した例文集から一番近い場面を探して提案します。 |
| 🎤 音声入力 | ブラウザ標準の音声認識（日本語 / English 切替）。話すだけで検索できます。 |
| 🔊 発音再生 | ブラウザ標準の音声合成で英文を読み上げ。通常 / 🐢ゆっくり を選べます。 |
| 🎙️ 発音練習 | 自分の声を録音して、お手本と聞き比べできます。 |
| 📚 場面別 | あいさつ・雑談 / あいづち・リアクション / 買い物・飲食店 / 旅行・道案内 / 仕事・ビジネス / 電話・オンライン / 困ったとき・聞き返し / 家族・日常生活 / 学校生活 / 体調・病院 / 天気・季節・行事 / 趣味・雑談のきっかけ / ほめる・お祝い・気遣い の13場面（約90例文）。子どもから大人・高齢の方まで、幅広い年代の日常会話をカバーしています。 |
| 🎯 復習クイズ | 日本語を見て「伝わる英語」を思い出すテスト形式。お気に入りがあればそこから出題。 |
| ⭐ お気に入り | 覚えたい表現をブックマーク。 |
| 🕘 履歴 | 調べた内容を自動で記録（端末内保存）。 |

- **完全無料 / ログイン不要 / 端末内保存**（データは `localStorage` に保存されます）
- **自動アップデート**: 例文追加などの改善版を公開すると、開きっぱなしのタブでも自動で最新版に切り替わります（入力中は奪わず、操作の切れ目で更新）
- **単一HTML**（フォント表示用に Google Fonts の CDN を利用。読み込めない場合はシステムフォントに自動フォールバックするため、オフラインでも機能自体は問題なく動作します）
- **ホーム画面に追加OK**: スマホのブラウザから「ホーム画面に追加」すると、専用アイコンでアプリのように開けます

## 例文の追加・編集（この枠組みの育て方）

例文はすべて `index.html` 内の `PHRASES` 配列にまとまっています。
**このオブジェクトを1つ追記するだけ**で例文を増やせます。

```js
{
  id:"g_sample",              // 一意なID（カテゴリ頭文字_名前 が目安）
  category:"greeting",        // greeting | reaction | shopping | travel | business | phone | help
  prompt:{ en:"...", ja:"..." },   // 相手のひと言 / 言いたいこと
  situation:"この場面と“意図”の説明",
  keywords:["...","キーワード"],    // 検索用（日本語・英語まぜてOK）
  school:{ en:"...", ja:"..." },   // 学校で習う形
  natural:{ en:"...", ja:"..." },  // 実際に伝わる形
  point:"なぜそう返す/言うのかの解説（<b>強調</b>可）",
}
```

- `keywords` を厚めに書くほど、自由入力で見つかりやすくなります。
- カテゴリを増やしたいときは `CATEGORIES` にも追記してください。

### キャラとの会話を増やす

`CHARACTERS`（キャラ一覧）と `CONVERSATIONS`（キャラごとの会話シナリオ）も同じファイル内にあります。
1キャラに複数の会話シナリオを持たせられるので、`CONVERSATIONS.emily` のように配列へ追記するだけで増やせます。

```js
{
  npc:{ en:"...", ja:"..." },           // キャラのセリフ
  choices:[
    { style:"school",  en:"...", ja:"..." },
    { style:"natural", en:"...", ja:"...", point:"なぜ自然なのかの解説" },
  ]
}
```

どちらの選択肢を選んでも次のターンに進み、最後のターンまで話し切れる構成です（AI不使用・完全無料）。

## 技術メモ

- 単一HTML（HTML / CSS / JS すべて `index.html` に内包）
- 音声入力: Web Speech API（`SpeechRecognition`）
- 音声出力: Web Speech API（`SpeechSynthesis`）
- 録音: `MediaRecorder` API
- 保存: `localStorage`
- フォント: Google Fonts（Zen Kaku Gothic New / Poppins）をCDN経由で読み込み
- 依存ライブラリなし（外部通信はフォント読み込みのみ）
- アイコン: `favicon.svg`（開いた本 + "English"の吹き出し + 会話中を示す黄色いドット吹き出しのロゴマーク）＋各サイズPNG（`favicon-16/32/48.png`, `icons/apple-touch-icon.png`, `icons/icon-192/512.png`）。`manifest.json` でホーム画面追加時のアイコン・アプリ名を指定
